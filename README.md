## üöÄ Featured Project: AI Lead Qualification Engine
**Stack:** `n8n` ‚Ä¢ `OpenAI GPT-4o` ‚Ä¢ `PostgreSQL` ‚Ä¢ `Twilio` ‚Ä¢ `Airtable`

An autonomous SMS receptionist designed to qualify service leads in real-time. This system replaces manual intake by using a "Verification Loop" to ensure data accuracy before triggering business alerts.

#### Key Engineering Highlights:
* **Persistent Memory:** Implemented a PostgreSQL-backed session manager to maintain conversation context across asynchronous SMS exchanges.
* **Logic Branching:** Engineered a parallel execution path that separates clean customer-facing chat from internal system signals (`[[SUMMARY_COMPLETE]]`).
* **Automated Hand-off:** Real-time data synchronization with Airtable and instant high-priority email notifications for "Ready-to-Quote" leads.
